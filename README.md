 AWS End-to-End Data Pipeline

 Project Overview
This project demonstrates an end-to-end data analytics pipeline built using AWS services. The pipeline ingests raw data into Amazon S3, processes it using AWS Glue, analyzes it using Amazon Athena, and visualizes insights in Amazon QuickSight.

  AWS Services Used

1.Amazon S3 – Data storage 
2.AWS Glue – create crawler
3.Amazon Athena – SQL-based analysis
4.Amazon QuickSight – Data visualization
5.IAM – Access control 


 Workflow

 1. Raw data is uploaded to S3 (raw zone)
 2. AWS Glue creates crawler (database)/(table)
 3. Athena queries are used to analyze the data
 4. QuickSight visualizes the insights

 Key Insights
 
  1.sum of profit by city and sub category
  2.sum of profit by city


 Learnings
  1.Building serverless data pipelines on AWS
  2.Querying S3 data using Athena
  3.Creating dashboards in QuickSight
